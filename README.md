**Black Bulls** is a premium, high-performance gym management dashboard designed for tracking subscriptions, member attendance, and financial growth. Built with a focus on privacy and speed for independent gym owners.

---

## ✨ Key Features

- **📊 Management Dashboard**: Real-time overview of active members, monthly revenue, and growth metrics.
- **👥 Member Management**: Full-featured system to add, edit, and track gym members and their subscription status.
- **📈 Financial Reports**: Detailed charts and data visualization for revenue tracking and membership trends.
- **🔐 Secure Access**: Protected admin portal powered by Supabase Authentication.
- **⚙️ Configurable Plans**: Easily manage subscription tiers, pricing, and gym settings.
- **⚡ Modern UI**: A sleek, dark-themed interface built with custom CSS for a premium aesthetic.

---

## 🛠 Technology Stack

- **Frontend**: React 19, Vite
- **Routing**: React Router 7
- **Icons**: Lucide React
- **Charts**: Chart.js & React-Chartjs-2
- **Backend**: Supabase (PostgreSQL + Auth)
- **Deployment**: Vercel

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Jen9x/GYM.git
   cd GYM
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Environment Setup**
   Create a `.env` file in the root directory and add your Supabase credentials:
   ```env
   VITE_SUPABASE_URL=your_supabase_url
   VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
   ```

4. **Run Development Server**
   ```bash
   npm run dev
   ```
